# Project 1 IF3260 Grafika Komputer
2D Web-Based CAD with WebGL

## Deskripsi Singkat
Program yang dibuat berupa suatu perangkat lunak computer-aided design (CAD) 2 dimensi yang dijalankan dalam sebuah website. Program ini dapat melakukan beberapa kemampuan CAD dasar antara lain membuat model garis, persegi, persegi panjang, dan poligon. Program juga dapat memberikan beberapa modifikasi terhadap bentuk-bentuk tersebut. Gambar yang telah dibuat dapat diekspor ke sebuah file .json untuk dibuka kembali di lain waktu.

## Instalasi & Cara Penggunaan
Jalankan ```index.html``` di web browser.
